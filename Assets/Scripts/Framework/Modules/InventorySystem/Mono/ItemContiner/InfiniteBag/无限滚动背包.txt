
流程：
{
	首先初始化：
	{
		获取背包容器接口

		订阅容器接口的更新事件

		实例化对应数量的网格格子
		{
			通过容器接口获取总数，得到总行数，如果少于实现无限滚动的的数量（一面 + 缓冲行）则至少实例化一面的格子，
			实例化
		}
		更新整个页面
		{
			遍历格子，更新格子位置，通过索引计算行列，再计算具体位置，并调整其位置
			遍历格子，再更新格子的数据，根据索引查询该索引的位置
		}
		订阅容器接口的物体更新事件
	}
	其次实现无限滚动背包
	{
		通过一个方法订阅ScrollRect的onValueChanged事件
		事件触发时开启一个异步函数，他会逐帧调用界面更新方法
		界面更新方法
		{
			计算当前的最顶部行数，计算滑动方向（通过记录一个lastTopRow，以及当前计算的nowRow的差值）
			得到差值，逐行逐行地添加前进方向的一行格子还有移除远离方向的一行格子
			
		}
	}

}

注意事项：
	背包在GridInit方法中的实例化格子方式决定了格子的背包形式只会选择固定整页的形式，而不是只有物体或者固定最少格子，而是会满满一整页的形式
